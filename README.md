
# Yat-OS

> 中山大学计算机学院《操作系统原理》课程教学演示操作系统

一个从零开始实现的 32 位 x86 操作系统，覆盖从 BIOS 启动到内核运行的全流程。

## 核心特性


**启动阶段**

1. **MBR 主引导记录** — 位于磁盘第 0 扇区，负责加载 Bootloader
2. **Bootloader** — 实模式初始化、LBA 硬盘读写、开启 A20 线、加载内核
3. **实模式 → 保护模式切换** — 加载 GDT、设置 CR0.PE 位、远跳转
4. **二级分页** — 建立页目录表与页表、开启 CR0.PG 位、虚拟地址映射

**内核初始化**

5. **GDT 段描述符** — 代码段、数据段、堆栈段、显存段
6. **IDT 中断描述符表** — 256 个中断向量、`lidt` 加载
7. **8259A PIC** — 可编程中断控制器、主从片级联、EOI 信号
8. **时钟中断与调度** — `int 0x20`、时间片轮转、线程切换

**内存管理**

9. **位图 (Bitmap)** — 1 bit 标识一个页面的使用状态
10. **地址池 (AddressPool)** — 以页为单位分配 / 释放物理内存
11. **内核空间 & 用户空间** — 物理页隔离、内核虚拟地址池

**程序管理**

12. **内核线程 (Thread)** — PCB 结构（4KB）、独立栈空间、上下文切换
13. **进程 (Process)** — 页目录表、独立虚拟地址空间、`fork` / `exit` / `wait`
14. **系统调用** — `int 0x80` 中断、调用表分发（共 5 个）

**同步**

15.  **自旋锁 (Spinlock)** — 基于 `xchg` 原子的互斥原语

## 目录结构

```
Yat-OS/
├── include/          # 头文件（内存、中断、线程、位图、地址池等）
├── src/
│   ├── boot/         # MBR、Bootloader、入口汇编
│   ├── kernel/       # 中断、内存、线程、程序加载、同步
│   └── utils/        # 位图、地址池、链表、标准库
├── build/            # 编译输出
├── run/              # 运行脚本
├── Makefile          # 编译配置
├── note.md           # 实验笔记
└── LICENSE           # MIT License
```

## 编译与运行

```bash
cd build
make          # 编译
make run      # 运行（Bochs/QEMU）
make clean    # 清理
```

**环境**：g++/gcc + NASM + ld + QEMU
