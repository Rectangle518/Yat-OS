
# Yat-OS

> 中山大学计算机学院《操作系统原理》课程教学演示操作系统

一个从零开始实现的 32 位 x86 操作系统，涵盖实模式保护模式切换、内存管理、中断处理、内核线程等核心概念。

## 核心特性

- **MBR + Bootloader** — 磁盘启动与实模式到保护模式切换
- **LBA 硬盘读写** — 通过 I/O 端口读写磁盘扇区
- **GDT/IDT** — 全局描述符表与中断描述符表
- **8259A PIC** — 可编程中断控制器
- **二级分页** — 页目录 + 页表，虚拟地址映射物理地址
- **位图 + 地址池** — 资源状态管理与页级内存分配
- **内核线程** — 线程 PCB、独立栈空间
- **并发与锁** — 基本的同步原语

## 目录结构

```
Yat-OS/
├── include/          # 头文件（内存、中断、线程、位图、地址池等）
├── src/
│   ├── boot/         # MBR、Bootloader、入口汇编
│   ├── kernel/       # 中断、内存、线程、程序加载、同步
│   └── utils/        # 位图、地址池、链表、标准库
├── build/            # 编译输出
├── run/              # 运行脚本
├── Makefile          # 编译配置
├── note.md           # 实验笔记
└── LICENSE           # MIT License
```

## 编译与运行

```bash
cd build
make          # 编译
make run      # 运行（Bochs/QEMU）
make clean    # 清理
```

**环境**：g++/gcc + NASM + ld + QEMU
